# Digital Hunters Landing Page

A modern futuristic gaming studio landing page built using pure HTML and CSS.  
This project focuses on creating a clean UI, responsive layout, glowing cyberpunk effects, and interactive hover animations inspired by modern game studio websites.

Preview includes:
- Hero section with animated character
- Project showcase cards
- Process workflow section
- Futuristic footer and CTA design

---

# Technologies Used

- HTML5
- CSS3
- Google Fonts (Orbitron + Inter)
- Remix Icons
- Font Awesome Icons

No frameworks or libraries were used for layout or styling.

---

# Features Implemented

### 1. Futuristic Hero Section
Large bold typography with glowing effects and animated central character image.

### 2. Responsive Flexbox Layout
Most sections use Flexbox for alignment and responsive positioning.

### 3. Smooth Hover Animations
Buttons, cards, icons, and navigation links include hover transitions and glow effects.

### 4. Animated Floating Hero Image
CSS keyframe animation creates floating movement for the hero character.

### 5. Project Showcase Cards
Interactive cards with scaling image hover effects and platform icons.

### 6. Process Workflow Section
Step-by-step workflow cards showing the development process visually.

### 7. Modern Footer Design
Footer contains CTA, contact information, social links, and branding.

### 8. Reusable CSS Components
Buttons, cards, wrappers, and utility styles are reusable and organized.
