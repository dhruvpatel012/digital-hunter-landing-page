# Digital Hunters Landing Page

A modern futuristic gaming studio landing page built using pure HTML and CSS.  
This project focuses on creating a clean UI, modern Flexbox layouts, glowing cyberpunk effects, and interactive hover animations inspired by modern game studio websites.

## Preview Includes

- Hero section with animated character
- Project showcase cards
- Process workflow section
- Futuristic footer and CTA design
- Modern Flexbox-based layout structure

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Google Fonts (Orbitron + Inter)
- Remix Icons
- Font Awesome Icons

No frameworks or libraries were used for layout or styling.

## Note

This project is currently not fully responsive.  
The main focus of this project was building a visually attractive futuristic UI using Flexbox positioning, glow effects, animations, and clean section layouts.

The design works best on desktop/laptop screens.  
Mobile optimization and advanced responsiveness can be added in future updates.
